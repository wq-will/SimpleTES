Architecture

How It Works

An evolutionary loop that grows stronger programs by sampling inspirations, generating candidates, evaluating them asynchronously, and checkpointing the best discoveries.

01
🌱

Seed Program

Start with a seed program, task prompt, and evaluator that together define the search objective.

02
🗃

Population Database

Keep a ranked database of programs, scores, and metrics so later prompts learn from strong runs.

03
🎯

Sample Inspirations

Sample strong and diverse inspirations using balance-style selection, with SEv2 extensions for chains.

04
🤖

LLM Generate

Send the task, inspirations, and metrics to an LLM so it proposes a stronger full program.

05
📈

Evaluate & Loop

Evaluate candidates, keep the best updates, and checkpoint progress for longer async searches.

↻ Steps 02 – 05 repeat until the score target is met or the generation budget is exhausted
