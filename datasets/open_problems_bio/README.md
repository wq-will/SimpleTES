# OpenProblems Benchmarks (SimpleTES Tasks)

Single-cell biology benchmark from [OpenProblems](https://openproblems.bio), set up as a SimpleTES task.

## Setup

```bash
cd datasets/open_problems_bio
bash setup.sh       # or: python scripts/prepare_task.py --task open_problems_bio
```

This creates `.venv/` with the bioinformatics stack pinned in `requirements.txt`, patches the OpenProblems v1.0.0 library for current scanpy / numpy via `openproblems_api_fix.sh`, and generates the pancreas denoising splits under `denoising/denoising_datasets/pancreas/`. Requires **Python ≥ 3.11**.

SimpleTES auto-detects the venv when running `datasets/open_problems_bio/denoising/`.

## Task: scRNA-seq Denoising

Given a raw UMI count matrix `X` (cells × genes), output a denoised count matrix `Y` of the same shape that minimises held-out reconstruction error.

**Metrics:**
- **MSE** in log-normalized space
- **Poisson negative log-likelihood** on raw counts (hard constraint)

### Files

| File | Purpose |
|------|---------|
| `denoising/init_program.py` | Starter program (evolved) |
| `denoising/evaluator.py` | Returns `{"combined_score": float, ...}` for a candidate |
| `denoising/denoising.txt` | Task prompt shown to the LLM |
| `denoising/denoising_datasets/pancreas/` | Bundled train/test splits (`pancreas_{train,test}_seed42.npy`) — generated by `setup.sh`, gitignored |

### Run

```bash
python main.py \
  --init-program datasets/open_problems_bio/denoising/init_program.py \
  --evaluator    datasets/open_problems_bio/denoising/evaluator.py \
  --instruction  datasets/open_problems_bio/denoising/denoising.txt \
  --model <your-model>
```

## OpenProblems patches

`openproblems_api_fix.sh` is applied automatically by `setup.sh` and fixes:

1. **Cache directory** — configurable via `OPENPROBLEMS_CACHE_DIR`
2. **CITE-seq loading** — dense arrays from `scprep` (`.tocsr()` fix)
3. **CZI API** — updated `tabula_muris_senis` loader
4. **`np.int` deprecation** — replaced with `int` in denoising utils
